# Student Result Manager 🎓

A beginner-friendly, terminal-based Python application designed to demonstrate how programming can solve simple, real-world problems—such as helping a teacher manage class grading records. 

## 🎯 Objective
The primary goal of this project is to build a foundational software tool that executes basic CRUD (Create, Read, Update, Delete) operations without relying on complex external databases or advanced frameworks. It serves as a practical introduction to Python logic, control flow, and data structures.

## ✨ Core Functionalities
The application provides a continuous, interactive menu that allows users to perform four primary actions:

* **Add Student:** Input a student's name and their corresponding numerical marks to register them in the system.
* **View Students:** Retrieve and display a complete list of all currently registered students and their marks.
* **Check Result:** Query a specific student's name to evaluate their score and determine if they passed or failed (based on a minimum passing threshold of 40 marks).
* **Exit:** Safely terminate the application.
